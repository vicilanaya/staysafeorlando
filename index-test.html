Human Trafficking

Stay Safe Orlando is an informative website for the citizens of Orlando, Central Florida. We hope you learn about human trafficking, how to identify its signs, how to report it, and how to get help if you are a victim. Please browse the website and share this resource with your community.

This project is brought to you by Code for Orlando, a chapter of Code for America. If you would like to engange your skills in civic action, please join your local brigade.
